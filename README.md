# Password Security Analyzer

Most people use passwords every day, but it is often difficult to know whether a password is actually secure or just feels secure. While learning about cybersecurity, I became interested in understanding what makes a password strong and how common password habits can create security risks.

Password Security Analyzer is a web application that evaluates password strength and provides real-time security feedback. It analyzes factors such as password length, character diversity, common password usage, predictable patterns, and overall password complexity to generate a security score and personalized recommendations.

The application also estimates password resilience, highlights potential weaknesses, and stores recent analyses locally to help users better understand and improve their password security practices.

I built this project to explore basic cybersecurity concepts while improving my frontend development skills using HTML, CSS, and JavaScript. It helped me understand how security-related logic can be translated into practical tools that provide meaningful feedback to users.
